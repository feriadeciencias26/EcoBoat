# EcoBoat Suquía

Sitio web multipágina para el proyecto "EcoBoat Suquía" — Barco Solar Recolector de Residuos — diseñado para la Feria de Ciencias en Jesús María, Córdoba.

## Archivos incluidos

- `index.html`
- `problema.html`
- `solucion.html`
- `investigacion.html`
- `impacto.html`
- `galeria.html`
- `acerca.html`
- `styles.css`
- `data.js`

## Qué contiene

- Navegación común en todas las páginas.
- Estilo compartido con Tailwind CSS y clases personalizadas en `styles.css`.
- Funcionalidad de menú móvil en `data.js`.
- Arquitectura multipágina con enlaces relativos.

## Cómo usar

1. Coloca todos los archivos en la misma carpeta.
2. Abre `index.html` en tu navegador.
3. Navega entre páginas usando el menú superior.

## Notas

- El CSS compartido incluye los colores de identidad del proyecto: `#005d90`, `#0077b6`, `#2c694e`, `#aeeecb` y `#f8f9fa`.
- `data.js` define datos del proyecto y controla el menú móvil.
